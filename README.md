# ESA-APHOA
1. A master-slave asynchronous communication architecture.
2. Hyper-heuristic scheduling framework.
3. A tower model for resource management.
4. The program entry point is main.m.
5. This experiment was run on the public computing power service platform(Linux) initiated and constructed by Hunan University of Science and Technology in 2023.
6. Using CEC2013 LSGO benchmark for testing.
